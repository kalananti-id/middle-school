Ayo belajar Flowchart!
Dalam sesi ini, kamu akan mengenal apa itu flowchart, simbol-simbol dasarnya, serta bagaimana menyusun langkah secara runtut dan logis.

Apa itu Flowchart?

Flowchart adalah diagram alur yang menunjukkan urutan langkah dari awal sampai akhir.

Contoh alur dan flowchart

START → Bangun tidur → Mandi → Sarapan → Berangkat sekolah → STOP

Simbol Dasar Flowchart

Terminator

Start / End
Contoh: “Mulai Belajar”

Process

Langkah / Proses
Contoh: “Mengerjakan PR”

Simbol Dasar Flowchart

Decision

Keputusan (Ya/Tidak)
Contoh: “Apakah hujan?”

Input/Output

Data masuk/keluar
Contoh: “Masukkan password”

Flow / Arrow

Arah alur
Contoh: “Lanjut ke langkah berikutnya”

Kenapa Flowchart Penting?

🧭

Berpikir Runtut

🗣️

Mudah Dikomunikasikan

Blueprint Sebelum Coding

Contoh Flowchart: Login Akun

🔎 Keterangan Singkat

Alur login memeriksa dua hal: (1) email sudah terdaftar, (2) password sesuai. Jika salah satu tidak terpenuhi, tampilkan pesan error; jika semua benar, berhasil masuk.

Ringkasan

🔑 Inti Materi

  • Flowchart = peta alur langkah.
  • Simbol dasar: Start/End (oval), Process (persegi), Decision (belah ketupat), Input/Output (jajargenjang), Flow (panah alur).
  • Membantu berpikir runtut & memudahkan coding.
