Memanggil Procedure (Call) üì£

Call procedure (dengan parameter)

Dipakai saat procedure membutuhkan input (misalnya x dan y).

// contoh: HitungDiskon(total, persen)
call HitungDiskon
x = 10000
y = 10%
Parameter itu seperti ‚Äúkolom yang harus diisi‚Äù saat procedure dipanggil.

Call procedure (tanpa parameter)

Dipakai saat procedure tidak membutuhkan input apa pun.

// contoh: BersihkanForm
to BersihkanForm do
set TextBox1.Text to ""
set TextBox2.Text to ""
call BersihkanForm
Cocok untuk reset tampilan, clear form, dan aksi rutin.

Konsep Input & Output üîÑ

Input (Parameter)

Nilai yang dikirim ke procedure agar bisa diproses.

  • Membuat procedure lebih fleksibel
  • Menghindari pengulangan blok
  • Bisa dipakai untuk nilai yang berbeda

Output (Hasil / Return)

Nilai yang dikembalikan procedure setelah memproses input.

Contoh

Blocks: Buat Procedure & Panggilnya üß±

A. Membuat Procedure

  1. Masuk tab Blocks ‚Üí Procedures
  2. Buat procedure baru: HitungLuas (gunakan parameter x dan y)
  3. Isi logika: x √ó y
  4. Pilih jenis result jika ingin mengembalikan nilai luas.
// procedure result
to HitungLuas (x, y)
result x * y

B. Memanggil Procedure

  1. Gunakan event: when ButtonHitung.Click
  2. Panggil: call HitungLuas
  3. Isi parameter: x dari TextBoxPanjang.Text, y dari TextBoxLebar.Text
  4. Set LabelHasil.Text menjadi hasil dari call procedure.
// event click
when ButtonHitung.Click
set LabelHasil.Text to
call HitungLuas(x=TextBoxPanjang.Text, y=TextBoxLebar.Text)
