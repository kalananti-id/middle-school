Review: Apa Itu Variabel?

Variabel adalah wadah untuk menyimpan nilai yang bisa berubah-ubah.

Jenis-Jenis Variabel di AppInventor

Ternyata, tidak semua variabel bisa diakses di seluruh bagian aplikasi! Ada dua jenis variabel berdasarkan jangkauan penggunaannya (scope):

Variabel Global (Yang Sudah Kamu Kenal)

Variabel Global seperti pengeras suara di sekolah

  • Bisa didengar di seluruh sekolah (semua bagian aplikasi)
  • Semua orang bisa mendengar pengumumannya
  • Pengumuman tetap ada sampai diganti dengan pengumuman baru

Contoh: Skor game, nama pemain, jumlah nyawa

Variabel Lokal (Yang Akan Kita Pelajari)

Variabel Lokal seperti berbisik di dalam kelas

  • Hanya bisa didengar di dalam kelas itu saja
  • Keluar kelas = tidak bisa dengar lagi
  • Bisikan hilang begitu pelajaran selesai

Contoh: Hasil perhitungan sementara, data yang sedang diproses

Aspek Global Variable Local Variable
Jangkauan Seluruh aplikasi Hanya dalam 1 blok
Umur Tetap ada sampai aplikasi ditutup Hilang setelah blok selesai
Blok initialize global initialize local
Kapan Pakai Data penting yang perlu diingat terus Data sementara untuk perhitungan

Variabel Lokal

Pengertian Lengkap

Variabel lokal adalah variabel yang:

  1. Hanya hidup di dalam satu blok kode (seperti di dalam when Button.Click)
  2. Tidak bisa diakses dari blok lain (blok lain tidak tahu variabel ini ada)
  3. Otomatis hilang begitu blok selesai dijalankan
  4. Dibuat dengan blok "initialize local" bukan "initialize global"

Di Mana Variabel Lokal Bisa Digunakan?

  • when Button.Click - saat tombol diklik (paling sering dipakai)
  • procedure - di dalam prosedur/fungsi yang kamu buat
  • for each item in list - di dalam perulangan
  • while loop - di dalam perulangan while
  • if-then-else - di dalam kondisi if

Siklus Hidup Variabel Lokal

Analogi Kertas Kasir yang Diperjelas:
Bayangkan kamu di kasir supermarket:

  • Kamu ambil kertas kecil untuk menghitung total belanja
  • Kamu tulis: "Roti 15.000 + Susu 8.000 = 23.000"
  • Kamu bayar 23.000 ke kasir
  • Kertas itu langsung kamu buang karena sudah tidak perlu

Kertas itu = variabel lokal! Hanya dipakai di kasir, setelah selesai langsung hilang. Kamu tidak bawa pulang kertas itu kan?

Mengapa Harus Hilang?

Alasan variabel lokal otomatis hilang:

  • Menghemat memori - tidak menyimpan data yang sudah tidak dibutuhkan
  • Mencegah kesalahan - blok lain tidak bisa mengubah nilainya secara tidak sengaja
  • Aplikasi lebih cepat - tidak ada data sampah yang menumpuk
  • Kode lebih rapi - setiap blok punya variabelnya sendiri

Mengapa Kita Membutuhkan Variabel Lokal?

  • Efisien: Tidak perlu membuat variabel global untuk data sementara
  • Rapi: Mencegah tumpang tindih nama variabel antar event
  • Aman: Nilainya tidak bisa diubah oleh bagian program lain di luar blok tersebut

Contoh Nyata:

Saat menghitung hasil dari dua input (harga & diskon), kamu hanya butuh menyimpan hasil perhitungan sementara. Nilainya tidak perlu diketahui oleh bagian program lain.

Kapan Menggunakan Global vs Lokal?

Cara Mudah Memilih Jenis Variabel

Tanya diri kamu 3 pertanyaan ini:

Pertanyaan 1: Apakah data ini perlu diingat terus?

YA ‚Üí Gunakan GLOBAL

Contoh: Skor game, nama pemain, level yang sudah dicapai

TIDAK ‚Üí Gunakan LOKAL

Contoh: Hasil perhitungan sementara, data input yang sedang diproses

Pertanyaan 2: Apakah data ini dibutuhkan di banyak tempat?

YA ‚Üí Gunakan GLOBAL

Contoh: Nama user (ditampilkan di banyak screen), pengaturan bahasa

TIDAK ‚Üí Gunakan LOKAL

Contoh: Total harga di 1 tombol, hasil konversi suhu

Pertanyaan 3: Apakah data ini berubah-ubah terus?

YA & perlu diingat ‚Üí GLOBAL

Contoh: Skor yang terus bertambah, timer countdown

YA & tidak perlu diingat ‚Üí LOKAL

Contoh: Angka random untuk 1 kali pakai, input sementara

Tips Mudah Mengingat:
Global = Seperti KTP (penting, dibutuhkan di mana-mana, disimpan lama)
Lokal = Seperti struk belanja (penting sebentar, setelah itu bisa dibuang)

Kesimpulan: Variabel Lokal adalah Sahabatmu!

  • Variabel lokal hanya hidup di dalam satu blok kode dan hilang setelah blok selesai dijalankan.
  • Menggunakan variabel lokal membantu membuat aplikasi lebih efisien, rapi, dan aman.
  • Gunakan variabel lokal untuk data sementara yang tidak perlu diingat lama atau diakses dari banyak tempat.

Ingat Kembali Analogi:
Variabel lokal seperti berbisik di dalam kelas atau kertas kasir sementara. Hanya dipakai sebentar, lalu hilang tanpa jejak!
