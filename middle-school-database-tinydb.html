Variable vs Database ü•ä

üì¶

VARIABLE

SEMENTARA

Seperti RAM. Cepat, tapi pelupa. Datanya ada saat aplikasi nyala, tapi langsung LENYAP saat aplikasi ditutup.

  • Hilang saat App Close
  • Contoh: Skor saat main
üóÑÔ∏è

DATABASE

PERMANEN

Seperti Hard Disk. Datanya ditulis kuat-kuat. Meskipun HP dimatikan, datanya TETAP ADA saat dibuka lagi.

  • Aman walau App Close
  • Contoh: Login, Highscore

Kenalan dengan TinyDB üêú

TinyDB adalah komponen Local Database di App Inventor. TinyDB menyimpan data langsung di dalam HP pengguna.

Cara Kerja (Tag & Value):

Bayangkan TinyDB seperti Loker Penyimpanan.

  • TAG
    Label di pintu loker. (Contoh: "NamaSaya")
  • VALUE
    Barang di dalam loker. (Contoh: "Budi")
